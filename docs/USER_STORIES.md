# User Stories

## Authentication

### Login

As a user,

I want to login into the system,

so that I can access my dashboard.


---

## Employee Management


As an HR Admin,

I want to add employees,

so that employee records are maintained.


---

## Employee Search


As a Manager,

I want to search employees by skills,

so that I can find suitable team members.


---

## AI Resume Analysis


As an HR Admin,

I want AI to analyze resumes,

so that I can identify suitable candidates quickly.


---

## Skill Recommendation


As an Employee,

I want skill recommendations,

so that I can improve my career growth.
